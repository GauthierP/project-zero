Introduction to the Embrace

Embrace is a smartband for seizure/epilepsy management provided by

It monitors the user's electrodermal activity and activity, and uses advanced machine learning to identify convulsive seizures and send alerts to caregivers. It also provides sleep, rest, and physical activity analysis.

The Embrace is set to work with the Alert App The Alert App which notifies the user's Caregivers through an automated SMS (text message) and phone call when Embrace detects a convulsive seizure.

There is also an optional Mate App which tracks the user's seizures and gives daily rest and physical activity reports, all automatically generated by the Embrace.
